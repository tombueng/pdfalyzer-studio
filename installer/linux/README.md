# Linux Installer

This directory contains scripts and resources for building a standalone Linux package for Pdfalyzer UI.

## Structure
- `assets/` — Icons, images, and other installer resources
- `scripts/` — Helper scripts for build automation
- `README.md` — Build and usage instructions

## Packaging Options
- AppImage
- .deb (Debian/Ubuntu)
- .rpm (Fedora/RedHat)

## Requirements
- Java runtime (to be bundled)
- Chromium binaries (to be bundled)
